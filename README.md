# Vityarthi-project-
A simple, interactive game where the player tries to guess a randomly generated number. The program provides feedback, telling the user if their guess is higher or lower than the secret number. 

#How to Play
Start the game: Run the program. It will generate a random number, usually between 1 and 100.
Make a guess: Enter a number when prompted.
Receive feedback: The program will tell you if your guess was too high, too low, or correct.
Keep guessing: Continue to guess until you find the correct number.
Win the game: Once you guess correctly, the game will congratulate you and may display the number of attempts it took. 

#Features
Generates a random number within a set range (e.g., 1-100).
User-friendly interface with input prompts.
Provides hints (higher/lower) to guide the player.
Keeps track of the number of attempts to win
